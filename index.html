O que é HTML?

Hyper Text Markup Language (Linguagem de Marcação de Hipertexto) é uma linguagem de marcação padrão desenvolvida para documentos projetados para serem exibidos em algum navegador web.

Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.
